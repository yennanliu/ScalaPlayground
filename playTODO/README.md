# ScalaTodo
> TODO board built via scala play framework

### Commands
```bash
#---------------------
# init project
#---------------------
sbt new playframework/play-scala-seed.g8
```

```bash
#---------------------
# run app
#---------------------
cd scalaPlayGround/playTODO/play-todo
sbt run

# visit http://localhost:9000/
```
